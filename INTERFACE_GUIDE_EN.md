# FBBP DB Bioinformatics Release Interface Guide

[English](./INTERFACE_GUIDE_EN.md) | [中文](./INTERFACE_GUIDE_CN.md)

## Purpose

Bioinformatics manuscript repository for data, schema, supplementary material, and reproducibility assets.

## Intended Readers

Journal editors, reviewers, readers, and future users of the release package.

## How To Read This Repository

- Use this repository for Bioinformatics Data availability and Code availability.
- Start from README.md, package_manifest.csv, and checksums_sha256.txt.
- Repository_Files_FBBP_DB_v1_release/ contains schema, tables, scripts, code, figure source data, and release metadata.
- Supplementary_Data_* folders mirror the manuscript supplementary data objects.

## Repository Boundary

This repository is an upload-ready public package. Local paths, runtime caches, logs, private raw data, model weights, and temporary working files were excluded before upload.
