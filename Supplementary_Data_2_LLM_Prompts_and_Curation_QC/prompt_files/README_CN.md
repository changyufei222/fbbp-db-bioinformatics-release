# Supplementary Data 2. LLM prompt 文件包说明

这不是编号 Supplementary Table，而是论文补充材料中的机器可读/可复现文件包。建议投稿时作为 ZIP 上传，或随 repository release 公开。

推荐放法：

1. 正文 Methods 只写 LLM-assisted curation 的流程和人工 QC。
2. Supplementary Table S3 只保留 compact QC summary，不放完整 prompt。
3. 完整 prompt 放入 Supplementary Data 2：`Supplementary_Data_2_LLM_prompts_20260608.zip`。
4. 如果期刊要求代码和数据仓库公开，也可把同一 ZIP 或文件夹放入 repository，并在 Data availability / Code availability 中引用。

这个文件包包含文献抽取 prompt template、专利抽取 prompt template、Extractor QC prompt、Reviewer QC prompt 和 Arbiter QC prompt。
