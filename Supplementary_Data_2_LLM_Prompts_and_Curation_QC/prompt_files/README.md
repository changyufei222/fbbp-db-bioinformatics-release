# Supplementary Data 2. LLM prompt files

This folder contains the full prompt files used for LLM-assisted literature/patent curation and three-role QC. These files should be submitted as Supplementary Data rather than as numbered Supplementary Tables.

Recommended submission format:

- Submit `Supplementary_Data_2_LLM_prompts_20260608.zip` as a supplementary data file.
- In Supplementary Table S3, refer to these files as the prompt archive.
- In Methods, describe the workflow as LLM-assisted curation with extractor, reviewer and arbiter roles.
- In Data availability or Code availability, cite the repository path if the journal prefers prompts to be released with code/data.

Prompt files included:

1. Literature extraction runtime template
2. Patent extraction runtime template
3. Extractor QC prompt
4. Reviewer QC prompt
5. Arbiter QC prompt
