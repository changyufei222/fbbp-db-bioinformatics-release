# Supplementary Data 2. LLM prompts and curation QC

This folder contains the full LLM prompt archive and compact machine-readable curation-QC outputs.

The CSV files in `llm_curation_qc_metrics_sanitized/` were copied from the release QC layer with absolute local filesystem paths removed or reduced to source basenames. Full prompt text is provided in `prompt_files/` and as a ZIP archive.
