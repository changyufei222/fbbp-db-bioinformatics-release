# FBBP DB Bioinformatics 投稿发布仓 界面说明

[English](./INTERFACE_GUIDE_EN.md) | [中文](./INTERFACE_GUIDE_CN.md)

## 这个仓库是做什么的

Bioinformatics 投稿用数据、schema、补充材料与可复现资产仓库。

## 谁应该先看这个说明

期刊编辑、审稿人、读者，以及后续使用该发布包的人。

## 仓库阅读顺序

- Bioinformatics 稿件中的 Data availability 和 Code availability 应引用这个仓库。
- 先读 README.md、package_manifest.csv 和 checksums_sha256.txt。
- Repository_Files_FBBP_DB_v1_release/ 包含 schema、tables、scripts、code、figure source data 和发布元数据。
- Supplementary_Data_* 文件夹对应论文补充数据对象。

## 上传边界

这个仓库是已经整理过的公开上传版本。上传前已经排除了本机路径、运行缓存、日志、原始私有数据、模型权重和临时工作文件。
