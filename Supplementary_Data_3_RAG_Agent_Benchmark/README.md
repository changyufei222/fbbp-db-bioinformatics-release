# Supplementary Data 3. RAG-agent benchmark

This folder contains the fixed benchmark set, model-result summaries, settings, controls, and repeatability outputs used for the RAG-agent evaluation.

## English-header aliases

Files ending in `_english_headers.csv` provide English column names for public reuse and journal-facing supplementary data checks. The original CSV files are retained as the raw benchmark exports.

Benchmark cell values, including original questions and expected-keyword wording, are intentionally preserved verbatim to make the reported evaluation reproducible. The alias map is provided in `Supplementary_Data_3_english_header_alias_map.csv`.
