# Supplementary Data 4. Figure source-data and code manifest

This folder contains the sanitized figure source-data/code manifest for manuscript Figures 1-5.

The manifest now separates figure artwork status from reproducible source data. For Figures 1-4, final manuscript artwork is submitted as figure artwork and is not duplicated in this Supplementary Data package; the supporting source tables are listed by package-relative path. For Figure 5, the PCSK9/1459D05 companion artwork and generation script are included under `Repository_Files_FBBP_DB_v1_release/figure_source_data/pcsk9_case_figure/`.

No local filesystem paths are required to resolve the package-internal source-data and code references.
