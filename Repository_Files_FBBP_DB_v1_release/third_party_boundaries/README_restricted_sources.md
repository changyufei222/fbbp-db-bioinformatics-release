# Third-party data and redistribution boundaries

This release candidate is designed to redistribute curated FBBP-DB records and derived analysis outputs, not raw third-party full-text corpora.

## Intended for public release

- Curated protein-level records and scaffold-family annotations.
- Active-loop / recognition-window boundaries and review status.
- Derived flexibility evidence fields and summary labels.
- Public identifiers such as PMID, DOI, PDB ID, UniProt accession, patent publication number, and source URLs.
- Derived structure/context features and analysis tables.
- Schema, metadata, and figure source data generated by the project.

## Not intended for redistribution

- Full-text journal PDFs or copyrighted article text.
- Bulk patent full-text dumps beyond public identifiers and derived evidence fields.
- Restricted third-party database raw exports, unless their license explicitly permits redistribution.
- API keys, LLM keys, local cache paths, private notes, or local absolute build paths.
- Raw temporary/intermediate files that include protected text or non-public processing artifacts.

Before public deposition, verify every source-specific license and keep only identifiers, provenance metadata, and derived project annotations where redistribution is uncertain.
