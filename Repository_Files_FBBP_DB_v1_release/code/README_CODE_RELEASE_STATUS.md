# Code release status

This data release candidate contains only code that was clearly located during this packaging pass:

- `code/strict_background/`: strict combined AIR/ALL background analysis scripts.
- `code/air_all_background/`: AIR/ALL comparison scripts.
- `code/binding_interface/`: binding-window/interface analysis scripts.
- `code/website_bundle/`: static website data-bundle build script.
- `scripts/validate_release.py`: lightweight release validation script generated for this package.

For Bioinformatics submission, a separate source-code repository should still be prepared and archived with Zenodo. That repository should include ETL/deduplication, structure-context selection, DSSP/SCOT processing, family-aware loop annotation, flexibility evidence generation, background comparison, binding-window analysis, figure generation, test data, and expected outputs.
