# Repository files for FBBP-DB v1 release

This folder collects the release objects that would normally be deposited in a data/code repository: release tables, schema files, analysis source-data, scripts, release manifest and checksums.

For the supplementary package, the figure source-data manifest was sanitized to remove local filesystem paths. The package-level `checksums_sha256.txt` in the parent folder is the checksum reference for this cleaned package.
